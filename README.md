# Python-OOP-Banking-System
# ATM Simulation System (Python OOP Project)

This is a simple banking system built using Python Object-Oriented Programming (OOP) concepts. It simulates basic ATM functionalities like account handling, deposit, withdrawal, and transaction history.

---

## Features

- Create and manage a bank account
- Deposit money
- Withdraw money with PIN verification
- Check account balance
- View transaction history
- PIN-based authentication
- Menu-driven system

---

## Concepts Used

- Classes and Objects
- Encapsulation
- Methods and instance variables
- Lists for transaction history
- Conditional statements and loops

---

## Technologies Used

- Python 3

---

## Project Type

Learning Project (Beginner to Intermediate OOP)
